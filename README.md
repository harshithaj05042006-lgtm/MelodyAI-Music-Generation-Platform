# 🎵 MelodyAI – AI Music Generation Platform

## 📌 Overview

MelodyAI is an AI-powered web application developed as **Task 3 of my CodeSoft Artificial Intelligence Internship**.

The platform enables users to generate original music using Artificial Intelligence based on different genres, moods, tempo, and instruments. It combines a modern user interface with AI-powered music generation to deliver an engaging user experience.

---

- 🎼 AI Music Generation
- 🎹 Multiple Music Genres
- 😊 Mood-Based Music Creation
- ⚡ Tempo Selection
- 🎸 Instrument Selection
- ▶️ Built-in Music Player
- 📜 Music Generation History
